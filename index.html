Enterprise IT Help Desk & Systems Administration Lab

Featured Project

Designed and deployed a complete enterprise Windows Server lab consisting of one Domain Controller and multiple domain-joined client computers. Technologies Used
Windows Server 2022 Active Directory DNS DHCP Group Policy PowerShell VirtualBox Windows 11 TCP/IP NTFS Permissions Shared Folders

Project Highlights

  • Designed and deployed a complete Windows Server 2022 enterprise lab from the ground up.
  • Installed and configured Active Directory Domain Services (AD DS) to manage users, groups, and organizational units.
  • Configured DNS and DHCP services to provide reliable name resolution and automatic IP address allocation.
  • Implemented Group Policy Objects (GPOs) to enforce password complexity and account lockout security policies.
  • Created secure shared folders using NTFS and Share permissions for controlled file access.
  • Mapped network drives and validated resource accessibility from domain-joined client computers.
  • Developed PowerShell scripts to automate user account management and routine administrative tasks.
  • Diagnosed and resolved enterprise IT issues involving DNS, DHCP, Group Policy, and file sharing using structured troubleshooting techniques.
  • Produced comprehensive deployment documentation, network diagrams, screenshots, and troubleshooting guides for the entire environment.
